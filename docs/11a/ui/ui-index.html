User Interface Issues

DLXS User Interface documentation describes both the scope of functions provided to the end user for collections implemented via the DLXS class middleware, and the means by which individual characteristics of collections which are instances of the class are configured.

A general overview of class functionality is provided in the DLXS Functionality Overview. The configuration of collection characteristics is achieved through a combination of HTML Templates, containing special processing instructions, and the ProcIns Module, which is responsible for resolving them into the HTML that is delivered to the user. HTML Templates / Interfaces gives a general overview of the components of interface configuration across all classes. Issues of configuration specific to a class are contained under individual class headings.
