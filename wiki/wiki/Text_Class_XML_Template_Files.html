Main Page > Mounting Collections: Class-specific Steps > Mounting a Text Class Collection > Working with the Text Class User Interface > Text Class XMl Template Files

The table below gives a brief description of all XML templates included in the Text Class distribution. Note that many of these files may have one or more XSL stylesheets associated with them.

accessoptions.xml Presents access options users denied access to collections. accessoptions.xsl, globalvariables.xsl
bbagcount.xml Small file containing the count of items in the bookbag to be come the source for the bookbag IFRAME. globalvariables.xsl, bbagcount.xsl, bookbagitemsstring.xsl
bookbagaction.xml Email, download, empty the bookbag.
bookbag.xml List of item sin the bookbag bookbagitemsstring.xsl, bookbag.xsl, bookbagheader.xsl, htmlhead.xsl, globalvariables.xsl, navheader.xsl, footer.import.xsl, searchforms.xsl, misc.xsl, xsl2htmlutils.xsl
browsecolls.xml List of all available collections withing their groups. bookbagitemsstring.xsl, browsecolls.xsl, htmlhead.xsl, navheader.xsl, footer.import.xsl, xsl2htmlutils.xsl, globalvariables.xsl
browse.xml Browse list for a given collection, dynamically generated. bookbagitemsstring.xsl, browse.xsl, browseheader.xsl, footer.import.xsl, globalvariables.xsl, htmlhead.xsl, navheader.xsl, xsl2htmlutils.xsl, xslfunctions.xsl, misc.xsl
clipviewer.xml Clip viewer user interface bookbagitemsstring.xsl, clipviewer.xsl, footer.import.xsl, globalvariables.xsl, htmlhead.xsl, misc.xsl, navheader.xsl, pageviewerheader.xsl, viewer.utils.xsl, viewer.xsl, xsl2htmlutils.xsl
dlxsclassglobals.xml Global xml at the class level.
dlxspageglobals.xml Global XML at the page level.
empty.xml Place holder.
footer.xml Standard footer.
header.xml Standard header. bookbagitemsstring.xsl, footer.import.xsl, globalvariables.xsl, header.xsl, htmlhead.xsl, misc.xsl, navheader.xsl, scopedivs.xsl, text.components.xsl, tocheader.xsl, xsl2htmlutils.xsl, xslfunctions.xsl
home.xml Home page for the class. bookbagitemsstring.xsl, globalvariables.xsl, navheader.xsl, home.xsl, htmlhead.xsl, footer.import.xsl
jsheader.xml For Javascript inclusion.
langmap.en.xml Lauguage map in English
langmap.fr.xml Language map in French.
loginoptions.xml Provides login options for users accessing restricted materials. loginoptions.xsl, globalvariables.xsl
navheader.xml Standard navigation header.
newmaterial.xml For informational material inclusion. newmaterial.xsl
pageviewer.xml Pageviewer user interface. bookbagitemsstring.xsl, footer.import.xsl, globalvariables.xsl, htmlhead.xsl, misc.xsl, navheader.xsl, pageviewer.xsl, pageviewerheader.xsl, viewer.utils.xsl, viewer.xsl, xsl2htmlutils.xsl
picklist.xml List of matching identifers for partial identifier submissions. bookbagitemsstring.xsl, picklist.xsl, picklistheader.xsl, htmlhead.xsl, navheader.xsl, footer.import.xsl, xsl2htmlutils.xsl, globalvariables.xsl
reslist.xml Results list, layers 1 and 2. bookbagitemsstring.xsl, footer.import.xsl, globalvariables.xsl, htmlhead.xsl, misc.xsl, navheader.xsl, reslist.xsl, results.xsl, resultsheader.xsl, scopedivs.xsl, xsl2htmlutils.xsl, xslfunctions.xsl
restricteditem.xml restricteditem.xsl, globalvariables.xsl
searchhistory.xml Search History page. searchhistory.xsl, bookbagitemsstring.xsl, htmlhead.xsl, globalvariables.xsl, navheader.xsl, footer.import.xsl, searchforms.xsl, results.xsl, xsl2htmlutils.xsl
search.xml Search forms. bookbagitemsstring.xsl, search.xsl, htmlhead.xsl, navheader.xsl, footer.import.xsl, searchforms.xsl, xsl2htmlutils.xsl, globalvariables.xsl, searchheader.xsl
text.xml Full text result for a section or entire text. bookbagitemsstring.xsl, footer.import.xsl, globalvariables.xsl, htmlhead.xsl, misc.xsl, navheader.xsl, scopedivs.xsl, text.components.xsl, text.xsl, textheader.xsl, tocheader.xsl, xsl2htmlutils.xsl, xslfunctions.xsl
viewer.xml Parent user interface for Pageviewer and Clipviewer
viewtextnote.xml Warning about downloading entire large works. viewtextnote.xsl, htmlhead.xsl
ww.xml WordWheel user interface. bookbagitemsstring.xsl, ww.xsl, htmlhead.xsl, globalvariables.xsl, navheader.xsl, footer.import.xsl, results.xsl, xsl2htmlutils.xsl
