[edit] Sitemaps

An application to generate sitemaps based on the specification at sitemaps.org is included at $DLXSROOT/bin/s/sitemaps.pl. This script creates a general site-wide sitemap at $DLXSROOT/web/sitemap.xml.gz which then links to collection specific sitmaps located in $DLXSROOT/web/sitemaps/. It can be used to create sitemaps for Text Class, Image Class and Findaid Class collections.

When run with a single collid parameter, sitemaps.pl will (re)generate a sitemap for a single collection. When run with no paramters sitemap.pl will attempt to (re)create sitemaps for all collections.

Currently the sitemap script is not connected to any collection building procedure and is intended to be run after indexing but prior to release. The use of sitemaps is intended to promote discoverabilty of DLXS content to search engines. Various search engines allow the registration of sitemaps in order to promote more efficient crawling of content, Goggle for instance offers Google Webmaster tools.
