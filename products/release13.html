DLXS Open Source Release 13

This code is provided as an Open Source resource. Please see the license statement in each file. It can be used with most versions of the Open Text search engine (frequently called "pat"), but is designed specifically for use with the XPAT search engine distributed through the UM Digital Library eXtension Service.

A freely available "Lite" version of the XPAT search engine may be downloaded below. It affords an opportunity to use DLXS middleware with smaller data sets (i.e., indexes may be built for source files up to 10 megabytes in size).
