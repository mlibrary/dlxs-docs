DLXS Open Source OAI Tools Package

This package consists of UMHarvester, UMProvider and OAITransform. This code is provided as an Open Source resource.

  • UMHarvester, v4-1-0:
    for harvesting OAI metadata from any OAI provider. (Also available on sourceforge.net)

  • UMProvider, v4-1-0:
    for providing OAI records from TextClass, ImageClass or FindaidClass. (Also available on sourceforge.net)

  • OAITransform, v4-1-0:
    for transforming OAI harvested metadata into DLXS Bibliographic Class, ready for indexing. (Also see the Bibliographic Class package for a complete listing)
