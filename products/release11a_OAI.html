DLXS Open Source OAI Tools Package

This package consists of UMHarvester and OAITransform. These tools interoperate with the DLXS Broker available as part of the full DLXS release. This code is provided as an Open Source resource. Please see the license statement in each file.

  • UMHarvester, v3-3-3:
    harvester software based on Ed Summers' N:O:H harvester APIs
  • OAITransform, v3-3-3:
    transformation tool to transform harvested oai_dc metadata into DLXS Bibliographic Class, ready for indexing
