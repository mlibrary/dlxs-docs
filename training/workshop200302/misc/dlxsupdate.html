DLXS: Recent Changes and Work Underway/Scheduled:

The most recent release of DLXS Middleware can be seen in the latest DLXS Release Notes: http://docs.dlxs.org/program/dlxs-releasenotes.xml

  • Main additions to Release 10 include:
    • TextClass:
      • Collections/groups interface improvements
      • Pageviewer support for continuous tone images
      • Results and document display abstracted from encoding level
    • ImageClass
      • Fixes to sorting, next/previous navigation, template fallback using DLXS lib code
  • Underway
    • XPAT
      • Investigate/implement Unicode support
    • TextClass
      • Header enhancements (flexibility of elements displayed)
      • Automated browse template creation
    • Finding Aids
      • Convert to class
    • Image Class
      • Investigate alternatives to Mr.Sid
      • Browsing, perhaps through database mechanisms
      • Basic support for other media types (sound, video)
    • General
      • Combine Text Class Bookbag and Image Class Portfolio mechanisms (so that Bookbags can be as persistent as Portfolios)

DLXS Users Group:

DLPS has initiated an informal DLXS user’s group.  In addition to an email group, this includes informal meetings in the spring and fall, probably in the same location as the DLF Forum (the first meeting was held in Seattle in November). We realize that not all of your institutions participate in DLF, but the chronological and geographic arrangement—i.e., alternating east and west coast, fall and spring—of the Forums should also make access to the user’s group meetings convenient for non-DLF members.

In addition, DLPS has solicited nominations for new developments in DLXS functionality. Suggestions for areas of work were received and clarified, and voting was done to order those priorities. The voting was not intended to select or eliminate work, only to prioritize it.  Each DLXS institution participated in the voting.  More information on the multi-voting process can be found at

http://web.mit.edu/tqm/multivoting.html
