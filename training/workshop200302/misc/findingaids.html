Finding Aids

The current implementation of Finding Aids is not a DLXS Class based implementation. There is no colldb, there is only a configuration file, in the "old school" style. Nevertheless, it uses XPat and many of the usual library modules.

  • Data
    • the EAD DTD
    • encoding practices differ from institution to institution; see the Bentley Historical Library's information on the EAD
  • Fining Aids Functionality
    • Search through finding aids and return results in a navigable form
  • Objects, modules used
    • XPat, SearchSet, XPatResultSet, ProcIns, DlpsSession
    • CGI
