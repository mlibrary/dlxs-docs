The Big Picture

An overview of how XPat and the DLXS Middleware work together to handle the retrieval and deliver of SGML data. See the Bar Napkin Overview.

  • Data Preparation
  • Middleware: translates user input into XPat query, retrieves SGML and delivers results wrapped in HTML

Architecture

  • See http://www.hti.umich.edu/index-all.html

Middleware Functionality:

  • cross collection searching
    • collection restrictions via categories in TextClass
    • groups in Bib and soon in others
  • cross machine searching
  • not cross class searching
  • results are not by default entire sections or works, but hits with navigation

Other

  • Server (we use Apache)
  • Authentication/Authorization
    • DLPS Apache module
    • Middleware only requires that a list of auth colls are put in the environment
  • xpat daemons to allow cross machine access between Middleware and Xpat
