Directory Structure

Under the DLXSROOT ...

  • web/
    • c/class (html pages, html templates, other web subdirectories)
      • gifcvtdir/ (must be owner nobody so it is writeable by the web server)
    • c/collection
      • images/ (symlink to directories in DLXSROOT for thumbnails and other
  • cgi
    • /c/class (middleware perl scripts, related files; e.g., subclasses, cfg files)
    • /c/class (middleware perl scripts, related files; e.g., subclasses, cfg files)
  • lib/ (perl modules)
  • obj/ (sgml files, related page images, pageview.dat)
    • c/collection/ (collection based organization)
    • a/b/c/abc1234/ (notis id/dlps id based organization)
    • lib/
      • db/ (colldb, groupsdb)
      • maps/ (TerminologyMapper files)
  • idx/
    • c/collection/ (XPat index files; e.g., collection.dd, collection.idx. collection.rgn)
      • WW/ (Xpat index files for collection's wordwheel)
  • bin/
    • c/class/ (pageview preparation scripts, etc.)
    • c/collection/ (collection preparation scripts)
      • WW/ (preparation scripts for collection's Wordwheel)

Virtual Hosts in Web Server

  • DLXSROOT environment variable
  • DocumentRoot, ScriptAlias

DLXS miscellaneous files & directories of note

  • /tmp/sessions/ & /tmp/sessionslock/ (Session data files and lock files. Directories and files must be owned by nobody so the web server can write to them.)
