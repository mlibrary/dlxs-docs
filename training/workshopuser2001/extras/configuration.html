Configuration

  • colldb
    • holds information about all collections in the Class
    • e.g., collection id, collection name, host machine, XPat dd location, etc.
  • groupsdb
    • combines certain collections into groups
  • cfg files
    • Class specific information: see textclass.cfg
    • Some general library modules have cfg files; e.g., DlpsSession.cfg
  • patservd, patviewd
    • need to edit access.c and re-make
    • we plan to pull certain information out and into a config file and simply deliver a binary

Note: For this workshop, participants have individual working versions of the colldb and groupsdb files
