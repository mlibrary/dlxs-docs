BibClass

  • Data
    • simple, bib class DTD, basically metadata is the item, indexed with XPat
    • Sample data
  • BibClass Middleware Components
    • bib-idx, bibclass.cfg, specific.functions.cfg, default.functions.cfg
    • colldb, groups, maps
    • SGML and XPat index
    • HTML templates
  • BibClass URL Parameters
    • Collection parameters: c, cc, g, bbc
    • Search types: type=simple|boolean
    • Pages: page=simple|boolean page=collpick page=history
    • Query parameters: q1, q2, ..., rgn1, rgn2, ..., op2, op3, ..., date1, date2
    • Views: view=reslist
    • Actions: bbaction=add|remove|download|email
    • Session ID: sid=106eb945036468c97458ee38d21186a0
    • BibClass-specific: fmt=short|long, sort=A-Z|Descending|Ascending,
