Finding Aids

The DLXS Version 10 implementation of Finding Aids is not a DLXS Class based implementation: there is no colldb, there is only a configuration file, in the "old school" style. Nevertheless, it uses XPat and many of the usual library modules. However, Version 11 will include a FindaidClass that will share a certain amount of the behavior seen in TextClass and whose base class searching and filtering will be based on the EAD2000 DTD.

Version 10 Finding Aids

  • Data
    • the EAD DTD
    • encoding practices differ from institution to institution; see the Bentley Historical Library's information on the EAD
  • Fining Aids Functionality
    • Search through finding aids and return results in a navigable form
  • Objects, modules used
    • XPat, SearchSet, XPatResultSet, ProcIns, DlpsSession
    • CGI
