Web Server Overview

We give here, as an example, the DLPS environment.

  • Apache (version 1.3.14 as of this workshop date)
  • Virtual hosts for various collections with their own specific
    • DocumentRoot
    • ScriptAlias
    • DLXSROOT environment variable
    • If no external Auth system, include AUTHZD_COLL environment variable containing a list of authorized collections (Middleware expects this environment variable to be set)
