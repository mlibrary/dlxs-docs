Other Systems Overview

  • daemon: A program (dlxsd) is packaged with DLXS software that allows the Middleware to work across machines (this might be an uncommon situation, but one we have here at DLPS). dlxsd uses one configuration file to allow access from particular domains, and another "verb" configuration file (where a verb is mapped to a particular program) that runs only those programs that the administrator sets up.
  • image conversion for Pageviewer functionality
    • tif2web: converts tiff files to gif files
    • c42pdf: converts tiff files to pdf files
  • MrSid (from LizardTech) for Image Class
    • stores files, using wavelet compression
    • retrieves images from these files at multiple resolution
