Future Directions

Ideas, and current plans for enhancements to DLXS include:

  1. Move sessions to other database types
  2. Combine Text Class Bookbag and Image Class Portfolio mechanisms (so that Bookbags can be as persistent as Portfolios)
  3. Combine usual region based searching with citation information based restrictions on the same page
  4. Make KWIC results optional and also available in region-type searches (e.g., boolean, proximity)
  5. More automatic browsing in Text Class
  6. Convert Finding Aids to a full-fledged class
  7. XPat
    1. Unicode support
    2. Regular expression searching
  8. More ...
