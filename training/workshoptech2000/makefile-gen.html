Makefile / Data Preparation Overview

(also see actual Makefile)
  • Principal Collection Targets
    • noded: runs nodefy (add NODE attributes to track DIV ancestry)
    • validate: runs nsgmls (validate SGML data)
    • norm: runs sgmlnorm (normalize SGML data)
    • dd (create XPat index for full text)
    • extra (create fabricated regions and indexes for them)
  • Related Data Targets (discussed later in Related Data)
    • pageview: runs makepageviewdata.pl (which creates pageview.dat files for page images)
    • wordwheel: runs Makefile for WW which runs makewordwheel.pl (create wordwheel derived SGML)
